🎓 What I Learned & Implemented in "Algorithms & Problem Solving - Level 3"
In this intensive course from Programming Advices, I deeply enhanced my problem-solving skills, logical thinking, and software architecture capabilities using C++. I transitioned from writing isolated code snippets to building complete, scalable, and secure console-based applications.

Here is a detailed breakdown of the advanced core concepts and capstone projects I mastered and implemented:

1. Advanced Matrix Operations (2D Arrays)
Matrix Generation & Manipulation: Generating random matrices, filling them with sequential numbers, and dynamic printing.

Summation & Calculations: Calculating the sum of rows, columns, and the entire matrix independently.

Matrix Transposition: Transposing matrices (flipping rows into columns and vice versa).

Matrix Comparisons: Checking if two matrices are equal, typical, or symmetrical.

Matrix Properties: Identifying identity matrices and scalar matrices, as well as counting specific numbers or intersections inside matrices.

2. Advanced String & Character Manipulation
Character Utilities: Converting characters to uppercase/lowercase and checking character types.

String Case Conversion: Converting entire strings to upper/lower case, and capitalizing the first letter of each word.

String Parsing & Tokenization: Counting words and spaces, splitting strings into vectors/arrays of words, and trimming whitespaces (Left, Right, and Full Trim).

String Joining & Reversing: Rejoining vectors back into a single formatted string, and reversing word orders within sentences.

Find and Replace: Replacing words or sub-strings inside a text using custom logic (case-sensitive and case-insensitive).

Data Cleaning: Stripping punctuation marks from text to clean raw data.

3. Capstone Projects (Real-World Applications)
🛠️ Project 1: Bank System (Core Modules)
A complete banking application designed to manage client data efficiently. Data is dynamically processed in memory and persisted using text files as a local database.

Show Client List: Displays all clients in a structured, formatted table layout.

Add New Client: Inserts new clients into the system with validation to prevent duplicate Account Numbers.

Delete/Update Client: Searches for a client and performs safe deletion or modification of records.

Find Client: Instantly retrieves and displays details of a specific client using their Account Number.

Transactions Menu: Implements functional deposit/withdraw modules and calculates total funds managed by the bank across all clients.

Technical Skills Applied: File handling (Read/Write), Record parsing/tokenization (Split/Join), Vectors for dynamic data management, and clean UI layout formatting.

🛠️ Project 2: Bank System with Users & Permissions
An enterprise-level extension of the Bank System that introduces custom security layers, user authentication, and granular access control.

Login Screen: Secured access requiring a valid username and password, with a strict limit on login attempts.

User Management (CRUD): A complete administration system to List, Add, Delete, and Update system users.

Permission-Based Access Control: Uses advanced integer/bit mask logic to grant or deny access to specific screens (e.g., denying a regular user access to the "Delete Client" or "User Management" modules).

Technical Skills Applied: Authentication logic, Advanced Logical/Bitwise Operations for permissions, and scalable structured programming architecture
